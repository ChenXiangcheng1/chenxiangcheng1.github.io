Flink

三大组件：DataSource 、Transformations、DataSInk输出

流处理：数据传输方式是当一条数据被处理完序列到缓存中立刻通过网络传输到下一个节点（流数据需要指定时间窗口）
批处理：所有数据处理完再传输

流处理API：StreamExecutionEnvironment、DataStream
批处理API：ExecutionEnvironment、DataSet

核心API：Table、SQL

安装配置

修改flink-conf.yaml

参数

修改master 为主节点主机名/IP

修改workers 为从节点主机名/IP

Flink集群的web界面：http://slave1:8081/#/overview

Flink ON YARN 在YARN中初始化的两种方式

第一种方式（yarn中常驻flink集群）：

bin/yarn-session.sh -jm 1024m -tm 1024m -d
yarn-session命令详解

bin/flink run ./examples/batch/WordCount.jar
找find yarn properties /tmp/.yarn-properties-root，知道applicationID，找yarn内Flink集群

yarn application -kill

第二种方式（yarn中临时flink集群） 依赖hapddot2.4.1：

bin/flink run -m yarn-cluster -yjm 1024m -ytm 1024m ./examples/batch/WordCount.jar 创建Flink集群提交任务
flink run命令详解

bin/flink cancel -yid applicationID Flink任务ID
